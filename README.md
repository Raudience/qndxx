# 青年大学习skip

**该项目旨在应付每周学生的青年大学习检查，以便节省你一周大约10分钟的时间和1%的手机电量。**

## 快速开始 | Quick start

1. 打开手机微信
2. 在微信中点开指定链接
3. 截图
